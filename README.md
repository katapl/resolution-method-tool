# Webová aplikace pro podporu výuky rezoluční metody
Aplikace je dostupná na adrese https://katapl.github.io/resolution-method-tool/.

Zdrojový kód je k dispozici ve veřejném repozitáři: https://github.com/katapl/resolution-method-tool.

Pro lokální spuštění je potřeba v kořenové složce projektu nainstalovat závislosti příkazem `npm install`. Následně aplikaci spustíte příkazem `npm run dev`.
